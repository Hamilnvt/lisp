# Lisp language in C (of course)

## Disclaimer

This implementation is (obviously) not complete and can change at any moment.

## How to compile

You will need [nob.h v3.2.1](https://github.com/tsoding/nob.h/releases/tag/v3.2.1) in one of your include directories

```bash
cc -o nob nob.c
./nob
```
## How to use

### REPL mode
```bash
./lisp
```

### INTERPRETER mode
```bash
./list <file.lsp>
```
